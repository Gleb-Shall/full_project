# Промпт для генерации сайта на Astro

## Контекст проекта

Создай современный, стильный одностраничный сайт на Astro для бизнеса "Лысинка" - салона профессионального бритья головы.

## Данные проекта

### Бизнес информация
- **Название**: Лысинка
- **Тип бизнеса**: Бритье головы
- **Индустрия**: Услуги
- **Локация**: улица геолокации йй
- **Цель**: Привлечь клиентов
- **Целевая аудитория**: Мужчины с доходом 100+к рублей

### Дизайн система

**Цвета:**
- Primary: #8B7A5D (основной цвет)
- Secondary: #9C917B (вторичный цвет)
- Accent: #2D2112 (акцентный цвет)
- Background: #ffffff (фон)
- Text: #62575B (текст)
- Дополнительные цвета: #0C100D, #463C30, #BB9C86, #6D5C3F, #DAB3A1, #9BA099, #A97361

**Типографика:**
- Шрифт заголовков: Inter
- Шрифт основного текста: Inter
- Размеры:
  - H1: 3rem
  - H2: 2.5rem
  - H3: 2rem
  - Body: 1rem

### Контент

**Hero секция:**
- Заголовок: "Лысинка - искусство бритвы"
- Подзаголовок: "Преобразите свой стиль с помощью профессионального бритья"
- CTA текст: "Связаться"
- CTA ссылка: #contacts

### Технические требования

- Язык: Русский (ru)
- Responsive: Да
- Анимации: Subtle (тонкие, ненавязчивые)
- Формы: Нет
- PWA: Нет

## Требования к дизайну

1. **Максимально стильный и современный дизайн**
   - Использовать премиум-эстетику, соответствующую целевой аудитории (мужчины с доходом 100+к)
   - Минималистичный, но выразительный дизайн
   - Элегантные переходы и анимации

2. **Структура сайта:**
   - Hero секция с заголовком и CTA
   - Секция "О нас" или описание услуг
   - Галерея изображений (4 фото)
   - Секция контактов

3. **Стилизация:**
   - Использовать все цвета из палитры гармонично
   - Применить шрифт Inter через Google Fonts
   - Создать плавные градиенты и тени
   - Использовать современные CSS техники (grid, flexbox, custom properties)

4. **UX/UI:**
   - Плавная прокрутка
   - Hover эффекты на интерактивных элементах
   - Адаптивная навигация
   - Красивая типографика с правильной иерархией

## Результат

Создать полноценный Astro проект с:
- package.json с необходимыми зависимостями
- astro.config.mjs с настройкой порта 8000
- Главной страницей index.astro
- Layout компонентом
- Стилями в отдельном файле или встроенными
- Компонентами для секций (Hero, Gallery, Contacts)
- Responsive дизайном для всех устройств
